The original base of the Scroll.club project. A platform where users can upload artworks and follow other artists. The dashboard allows users to view the newest work uploaded by the artists they followed. Users can also like and comment on artworks.

## Features

- User artwork upload functionality
- Artist following system
- Personalized dashboard showing latest works from followed artists
- Like and comment system for artworks
- Social networking for artists

## Technology Stack

This project is built using:
- **Vue.js** for the frontend interface
- **Laravel** for the backend API
- **MySQL** for database management

## Implementation Details

The platform serves as a social network for artists, allowing them to share their work and build a following. The dashboard provides a curated feed of content from artists that users follow, similar to social media platforms but focused specifically on artwork.

The like and comment features enable community engagement and feedback, helping artists connect with their audience and receive input on their work. This project laid the foundation for the more exclusive Scroll.club platform.
