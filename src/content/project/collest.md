A collaborative-learning treasure-hunting multiplayer game, made with Unity (C#). The game uses REST API to store and retrieve data. JSON Web Token (JWT) is used to secure user data.

## Features

- Multiplayer treasure-hunting gameplay
- Collaborative learning mechanics
- REST API integration for data management
- Secure authentication using JWT
- Real-time data synchronization
- Educational gameplay elements

## Technology Stack

This project is built using:
- **Unity** game engine
- **C#** for game logic and mechanics
- **REST API** for backend communication
- **JWT (JSON Web Token)** for secure authentication

## Implementation Details

COLLEST. combines gaming with education in a multiplayer treasure-hunting experience. The game implements collaborative learning mechanics that encourage players to work together while exploring and discovering treasures.

The technical architecture includes:
- **REST API Integration**: The game communicates with a backend server to store and retrieve game data, player progress, and shared information
- **JWT Authentication**: Implements secure user authentication and session management using JSON Web Tokens
- **Multiplayer Functionality**: Enables real-time interaction between players
- **Data Persistence**: Game progress and user data are stored securely on the server

The combination of educational content with engaging treasure-hunting gameplay creates an immersive learning experience. The secure backend ensures that user data and progress are protected while allowing seamless multiplayer interactions.
