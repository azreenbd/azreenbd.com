A platform to manage events. Users can create new events, register committee members and attendees. Made purely with HTML, CSS and PHP.

## Features

- Event creation and management
- Committee member registration
- Attendee registration system
- User-friendly interface
- Complete event tracking

## Technology Stack

This project is built using:
- **HTML** for structure
- **CSS** for styling
- **PHP** for backend logic and database operations

## Implementation Details

This event management system provides a straightforward solution for organizing and managing events. Event organizers can create events, designate committee members to help with organization, and track attendee registrations.

The system is built with pure PHP without any frameworks, demonstrating fundamental web development skills and server-side programming. The simple tech stack makes it easy to deploy on any standard web hosting service that supports PHP.

The platform handles the complete lifecycle of event management, from creation to attendee tracking, making it suitable for conferences, workshops, meetups, and other organized gatherings.
