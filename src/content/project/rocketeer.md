A simple Android game with tap-only control, made with Unity (C#). Art and design elements are self-made.

## Features

- Simple tap-only controls for easy gameplay
- Custom-made art and design elements
- Mobile-optimized for Android devices
- Engaging arcade-style gameplay
- Original graphics and assets

## Technology Stack

This project is built using:
- **Unity** game engine
- **C#** for game logic and mechanics
- **Android** platform targeting

## Implementation Details

Rocketeer is a mobile game designed with accessibility in mind, featuring intuitive tap-only controls that make it easy for anyone to pick up and play. The game is built entirely in Unity, leveraging the engine's powerful 2D capabilities and Android deployment features.

All art and design elements were created from scratch, showcasing both game development and graphic design skills. The tap-based control scheme is perfect for mobile devices, allowing players to control the game with just one finger.

The game demonstrates proficiency in Unity development, C# programming, mobile game optimization, and complete game asset creation from concept to final product.
