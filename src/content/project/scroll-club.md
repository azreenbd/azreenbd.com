A platform for artists to showcase their artwork and interact with each other. This platform also uses an invite code system to only allow club members to create a profile and upload their work.

## Features

- Exclusive artist community with invite code system
- Profile creation and management for club members
- Artwork upload and showcase functionality
- Artist interaction and networking capabilities
- Member-only access control

## Technology Stack

This project is built using:
- **Vue.js** for the frontend interface
- **Laravel** for the backend API
- **MySQL** for database management

## Implementation Details

The platform implements a secure invite-only system that ensures only approved members can join and participate in the community. Artists can create their profiles, upload their work, and engage with other members in a curated environment.

The invite code system adds an extra layer of exclusivity, making the platform ideal for private art clubs, professional networks, or selective creative communities.
