You're looking at it. This portfolio is a static site built with **Astro**,
using **Radix UI** primitives styled with **Tailwind CSS** for the interactive
bits, and deployed automatically to **GitHub Pages**.

## Highlights

- Content-driven: projects and blog posts are just markdown files
- Light/dark theme with no flash of the wrong theme
- Fully static output, deployed via GitHub Actions

## Stack

**Astro** + **React islands** (**Radix UI**) + **Tailwind CSS v4**.
