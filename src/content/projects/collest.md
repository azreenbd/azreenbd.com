Collest. is a multiplayer, treasure-hunting game built around collaborative
learning. Players work together to complete challenges, with game state synced
through a REST API.

## Highlights

- Collaborative-learning multiplayer gameplay
- REST API backend for storing and retrieving data
- JSON Web Tokens (JWT) securing user data

## Stack

Game client in **Unity / C#**, backend in **PHP** exposing a **REST API**
secured with **JWT**.
