An event management platform for creating and running events end to end.
Organisers can spin up a new event, register committee members, and manage
attendee sign-ups.

## Highlights

- Create and configure events
- Register committee members and attendees
- Simple, dependency-free PHP backend

## Stack

Built purely with **HTML**, **CSS** and **PHP**.
