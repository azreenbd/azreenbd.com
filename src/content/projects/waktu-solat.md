Waktu Solat is a progressive web app that shows Islamic prayer times sourced
from JAKIM's official e-Solat API. Because it's a PWA, it can be installed to a
phone's home screen and used like a native app.

## Highlights

- Live prayer times from JAKIM's e-Solat API
- Installable PWA that works offline once loaded
- Clean, mobile-first interface

## Stack

Built with **Vue.js** as an installable **progressive web app**.
